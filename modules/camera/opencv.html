4.3 安装OpenCV

4.3 安装OpenCV

有两种方式安装OpenCV。

1. 在虚拟环境中 pip install opencv-python

2. 安装.whl文件，网上能搜到。

操作过程

激活虚拟环境

cd /home/pi/.virtualenvs

source /gp37/bin/activate

pip安装opencv

pip install opencv-python

检测

python3
import cv2

注意事项

1. 安装期间的问题

我在安装完成后出现了以下错误 ImportError: libcblas.so.3: cannot open shared object file: No such file or directory

尝试pip install libcblas-dev，但是PyPif里找不到这个包。

所以我在全局安装了其他依赖，一个个试，但并不知道问题出在哪里。

sudo apt-get install libcblas-dev
sudo apt-get install libatlas-base-dev
sudo apt-get install libjasper-dev

sudo apt-get install libqt4-test
sudo apt-get install libqtgui4

sudo apt-get install libhdf5-dev
sudo apt-get install libhdf5-serial-dev

sudo apt-get update

总之，问题解决。

2. 安装拓展

pip install opencv-contrib-python

至于为什么要安装，目前还不清楚。

我自己目前还没有安装。
