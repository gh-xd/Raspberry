4.6 接收端py文件

4.6 接收端py文件

接收端和发送端（树莓派）所需要安装的python包一样。 可以在接收端创建一个与发送端一模一样的虚拟环境。

以下代码在windows或linux上都可以运行。

import cv2
import zmq
import base64
import numpy as np

# 实例化用来接收帧的zmq对象
# 注意此处的端口号要和发送端（树莓派）上.py内设置的一样
context = zmq.Context()
footage_socket = context.socket(zmq.PAIR)
footage_socket.bind('tcp://*:5555')

while True:
    frame = footage_socket.recv_string() # 接收到数据
    img = base64.b64decode(frame) # 解码并保存到缓存中
    npimg = np.frombuffer(img, dtype=np.uint8) # 将解码的内容转化为一维数组
    source = cv2.imdecode(npimg,1) # 将一维数组解码为图像
    cv2.imshow("Stream", source) # 显示图像
    cv2.waitKey(1) # 一般在imshow()后摇使用waitKey()，给图像绘制留下时间，不然窗口会出现无响应的情况
